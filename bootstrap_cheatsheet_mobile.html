What Bootstrap Is

Bootstrap is a front-end framework that helps build responsive interfaces faster with a grid system, prebuilt components, and utility classes.

Remember: Bootstrap = grid + components + utilities + responsive design.

Grid Basics

<div class="container"> <div class="row"> <div class="col-12 col-md-8">Main</div> <div class="col-12 col-md-4">Sidebar</div> </div> </div>
  • container wraps and centers content
  • row creates a grid row
  • col-* controls column width
  • col-md-8 starts at the medium breakpoint

Bootstrap vs CSS vs JavaScript

BootstrapLayout, components, utilities, responsiveness.
Plain CSSFull custom styling and design control.
JavaScriptBehavior, logic, and interactivity.

Good Fits

  • Prototypes and MVPs
  • Dashboards and internal tools
  • Interfaces that reuse cards, navbars, forms, and alerts

Less Ideal Fits

  • Highly custom brand-heavy sites
  • Projects built around a unique design system
  • Teams that prefer a utility-first workflow like Tailwind
